## 项目介绍

  * **前台**：Spring Boot 支持的模板引擎 Thymeleaf + jQuery 完成页面信息展示
  * **后台**：Spring MVC + Spring Boot + MyBatis 框架 进行整合开发

## 系统开发及运行环境

  * 操作系统：Windows

  * Java开发包：JDK 8
  * 项目管理工具：Maven  3.6.0
  * 项目开发工具：IntelliJ IDEA
  * 数据库：MySQL
  * 缓存管理工具：Redis  3.2.100
  * 浏览器：谷歌浏览器
